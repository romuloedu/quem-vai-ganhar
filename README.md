# Quem vai ganhar a Copa?

**Previsões estatísticas para a Copa do Mundo de 2026, calculadas antes de cada apito inicial.**

Acesse: [romuloedu.github.io/quem-vai-ganhar](https://romuloedu.github.io/quem-vai-ganhar)

---

## O que você vai encontrar

Uma página que transforma dados em previsões reais para a Copa de 2026. Antes de cada jogo, o modelo calcula a chance de vitória, empate e derrota. Conforme os resultados chegam, você acompanha onde as previsões acertaram e onde o futebol surpreendeu.

Há também uma análise completa do torneio: quem são os favoritos ao título, as chances do Brasil em cada fase, a comparação entre continentes e até a probabilidade de um clássico Brasil x Argentina na final.

## Como foi feito

O modelo usa regressão logística treinada com histórico de seleções, ranking FIFA e desempenho recente. Para as probabilidades de título e avanço de fase, o torneio inteiro foi simulado 100 mil vezes usando o método de Monte Carlo.

Nada de palpite. Distribuição de probabilidade.
